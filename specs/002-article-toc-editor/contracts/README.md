# Contracts: 文章阅读与写作页面目录与编辑重构

**Branch**: `002-article-toc-editor`  
**Related Spec**: `specs/002-article-toc-editor/spec.md`

本功能主要为现有 Next.js Web 应用内部的页面与组件改造，不新增对外暴露的独立 API 或协议层接口，因此目前**没有单独的外部合约文件**。

后续如引入以下能力时，可在本目录内补充具体契约说明（如 OpenAPI、事件格式、组件接口文档等）：

- 为文章目录或写作功能提供独立 HTTP/JSON API；  
- 对接第三方内容平台或编辑器服务；  
- 将编辑器封装为可复用的独立包或组件库，对外暴露稳定接口。
